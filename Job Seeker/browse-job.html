Senior Frontend Developer

TechCorp Inc.

•

San Francisco, CA

Full-time $120,000 - $150,000 per year 5+ years experience

Job Description

We're looking for an experienced Frontend Developer to join our team. You'll be responsible for building responsive web applications using React, TypeScript, and modern CSS frameworks.

Responsibilities

  • Develop new user-facing features using React.js
  • Build reusable components and front-end libraries for future use
  • Translate designs and wireframes into high-quality code
  • Optimize components for maximum performance across a vast array of web-capable devices and browsers
  • Collaborate with the design team to implement visual elements

Requirements

  • 5+ years experience with JavaScript and React.js
  • Strong proficiency in TypeScript
  • Experience with responsive design and CSS frameworks like Tailwind CSS
  • Familiarity with RESTful APIs and GraphQL
  • Understanding of server-side rendering and its benefits
  • Experience with state management libraries (Redux, MobX, etc.)
  • Knowledge of modern authorization mechanisms, such as JWT

Benefits

  • Competitive salary: $120,000 - $150,000 per year
  • Health, dental, and vision insurance
  • 401(k) matching
  • Flexible work hours and remote work options
  • Professional development budget
  • Generous paid time off

Required Skills

React TypeScript Tailwind CSS Redux REST APIs GraphQL
